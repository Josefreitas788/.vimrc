# .vimrc

Para instalar os plugins siga os seguintes passos:

1 - mova o arquivo .vimrc para o $HOME   

`$ mv .vimrc ~/`

2 - `$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

3 - Inicie o vim e digite `:PluginInstall`
